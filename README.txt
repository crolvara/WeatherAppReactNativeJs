# WeatherApp

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the project: `expo start`
4. Run tests: `npm test`
5. Make sure your devices are connected on the same Wifi

## Description

A simple React Native app that displays the current temperature in Celsius, Fahrenheit, and Kelvin for a user-selected city. The app includes a loading state while fetching the data and uses animations to enhance the user interface when displaying the weather information.

## Technologies

- React Native
- Expo
- Axios
- Jest
- OpenWeather API
- Reanimated

## Features

- Search input for entering the city name.
- Display area for showing the current temperature in Celsius, Fahrenheit, and Kelvin.
- Loading spinner while fetching weather data.
- Smooth fade-in animation for displaying the weather information.
- User-friendly error messages.
